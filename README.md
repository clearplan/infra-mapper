# ClearPlan InfraMapper

A standalone, single-file infrastructure mapping and digital transformation roadmap tool — no installation, no server, no dependencies.

Download `infra-mapper.html` from the [latest release](https://github.com/clearplan/infra-mapper/releases/latest) and open it in any modern browser.

---

## Features

**Network Diagram**
- Map your current infrastructure as a node graph with categories, connections, and icons
- Future State view overlays planned upgrades on the current diagram
- 200,000+ icons via Iconify; upload your own logos per node
- Auto layout, search/highlight, undo/redo, custom themes

**Roadmap**
- Gantt-style timeline grouped by swimlane (category)
- Auto-phases upgrade targets based on dependencies
- Add manual tasks for support, audits, or other non-node activities
- Drag bars to adjust start dates and durations; set hard deadlines
- Export to PNG or multi-page PDF

**Data**
- All data lives in a single JSON file — save, load, and share with your team
- Auto-saves to browser local storage as a backup
- No accounts, no cloud sync, no telemetry

---

## Usage

1. Download [`infra-mapper.html`](https://github.com/clearplan/infra-mapper/releases/latest) from the latest release
2. Open the file in Chrome, Edge, Firefox, or Safari
3. Use **Save** to export your work as a `.json` file
4. Reload from any machine using **Load**
